For windows

1. Install VisualStudio Build tools, Qt6, and cmake 

2. Make sure you set your env variables for Qt 
    a. Qt6_DIR=path\to\qt\install\<toolchain>
    b. QT_PLUGIN_PATH=path\to\qt\install\plugins

- install openCV to c:\opencv\

3. Run the build with the camerasdk path as the first argument 
./WinBuild.bat <Path-to-camera-sdk-path>
